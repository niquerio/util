# Util

This is a repo for cli stuff I find useful.

## Prerequisits

* poetry
* python3
* gh

## Installation

To set this up run:

```
poetry install
```

Then copy the example config.example.yaml to config.yaml. Edit to add the repos you want to check.

```
cp config.example.yaml config.yaml
```

Then:

```
poetry run python util.py
```
